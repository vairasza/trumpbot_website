Rules for the Donald Trump Tweet Generation Bot:

  • In the top left corner you can choose between 2 versions of the Trumpbot: version 1.0 gets his topics through clustering with KMeans and version 2.0 gets his topics through matching predefined topics with regular expressions.

  • Type in a sentence with a topic like "Barack Obama" or "China" in natural language and the Trumpbot will answer with a tweet.

  • Type "!help" if you want to get a list of available topics!

  • The returned tweet is either a generated tweet by the Trumpbot or a real tweet by @realdonaldtrump.

  • React with "!fake" on a returned tweet if you think it is fake and generated by the Trumpbot.

  • React with "!real" on a returned tweet if you think it is real and is from @realdonaldtrump.

  • Click the Help Button in the left corner if you want to get help with commands for the Trumpbot.

Commands for Trumpbot:

  • "!help" -- output all topics that the Trumpbot knows

  • "!fake" -- react to a returned tweet if you think that it is a fake tweet generated by the Trumpbot

  • "!real" -- react to a returned tweet if you think that it is a real tweet from @realdonaldtrump
